# 📊 Stock Market Risk & Return Analysis

## 🚀 Project Overview

Developed an end-to-end financial analytics solution to evaluate stock performance, measure investment risk, and identify optimal risk-return opportunities using Python, SQL, and Power BI.

This project simulates real-world financial analysis used by analysts for portfolio decision-making.

---

## 🎯 Business Objective

* Analyze stock performance across multiple companies
* Measure risk using volatility
* Compare return vs risk for investment decisions
* Identify top-performing and high-risk stocks
* Enable data-driven portfolio optimization

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* SQL (MySQL)
* Power BI (Dashboard Visualization)

---

## ⚙️ Data Processing & Feature Engineering (Python)

* Extracted stock data using yfinance
* Cleaned and structured raw financial data
* Created key features:

  * Daily Returns
  * Moving Averages (MA_10, MA_50)
  * Volatility (Risk)
  * Risk Categories (Low / Medium / High)

---

## 🧮 Financial Metrics Calculated

* Total Return
* Average Return
* Volatility (Risk)
* Sharpe Ratio (Risk-adjusted return)

---

## 🗄️ SQL Analysis (KPI Extraction)

Used SQL to:

* Identify top-performing stocks based on returns
* Analyze high-risk stocks using volatility
* Compare return vs risk across stocks
* Generate business-level KPIs for decision-making

---

## 📊 Dashboard Insights (Power BI)

### Key Visualizations:

* 📈 Stock Price Trend Over Time
* 🏆 Top Performing Stocks (Total Return)
* 📊 Moving Average Comparison (MA_10 vs MA_50)
* ⚖️ Risk vs Return Analysis (Bubble Chart)

### Key Insights:

* Higher return does not always imply higher risk
* Some stocks provide stable returns with lower volatility
* Moving averages help identify long-term trends
* Risk-return analysis supports portfolio optimization
